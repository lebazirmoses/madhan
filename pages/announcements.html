Web Development Hackathon

Welcome to the Frozen Hackathon Web Development Challenge! Put your coding skills to the test and build a creative and functional website using HTML, CSS, and JavaScript.

Requirements:

  • Your website must have at least three pages (e.g., Home, About, Contact).
  • Implement responsive design to ensure compatibility with various devices.
  • Use CSS styling for an attractive and cohesive layout.
  • Incorporate JavaScript for interactive features (e.g., form validation, image sliders).

Judging Criteria:

  • Functionality (30%) - How well does the website perform its intended functions?
  • Design (30%) - Is the website visually appealing and well-designed?
  • Code Quality (20%) - Evaluate the cleanliness and efficiency of the code.
  • Creativity (20%) - Assess the originality and innovation in the design and features.

Ready to showcase your skills? The contest begins on February 05, 2024. Stay tuned for more updates!
